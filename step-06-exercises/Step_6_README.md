# Step 6 — Exercises

Three short exercises practicing inline events other than `onclick`.
Each exercise has:

## Exercise 1 — `onmouseover` / `onmouseout`
Hover over the cards. Each card should swap its background colour when the
mouse enters, and return to normal when the mouse leaves.
Practices: `onmouseover`, `onmouseout`, `classList.add/remove`.

## Exercise 2 — `onfocus` / `onblur`
Click into each form field. The field should highlight (add a class) when
focused and un-highlight when blurred.
Practices: `onfocus`, `onblur` on `<input>` and `<textarea>`.

## Exercise 3 — `onload`
When the page finishes loading, automatically show a welcome message and
the current date in a banner at the top of the page.
Practices: `onload` on `<body>`, calling a function on page load.
